Color System

Pairing Rules

Do

  • Spring Green background + Science Blue text
  • Matte Blue background + Praxeti White text
  • Three-column split: Inch Worm | Light Blue | Science Blue
  • Gradient transitions within the same color family

Don't

  • Green text on blue background (low readability)
  • Manz on Praxeti White (insufficient contrast)
  • More than one extended palette color per composition
  • Full black #000000 — use Dark Steel or Midnight Mirage

06

Visual Motif

Pixel Dispersion

The signature visual treatment. Real-world objects dissolve into scattered pixel blocks at their edges — representing the transformation of physical value into digital infrastructure.

What It Communicates

Transformation

Physical value — cash, coins, organic growth — being converted through technology

Bridging

Real-world assets moving into cross-chain DeFi infrastructure

Movement

Value doesn't disappear — it disperses and reassembles elsewhere

Subject Categories

CategoryExamplesUse When
Organic GrowthFlowers, plants, leaves, botanical elementsYield, growth, earning potential narratives
Currency / CashDollar bills, hands holding moneyDeposits, capital deployment, financial action
Crypto AssetsBitcoin coins, tokensCross-chain context, crypto-native audiences
07

Visual Motif

How to Apply

ParameterGuideline
Pixel block sizeVary from large (near the object) to small (at the dispersion edge)
DirectionOutward from the object, typically upper-right or trailing edge
Color extractionPixel blocks sample colors from the source — pinks from flowers, greens from leaves, golds from coins
BackgroundClean white or solid brand color blocks — never busy
BlendingPhotorealistic → grid-aligned pixels → scattered single pixels
CoverageDispersion affects 20–40% of the object. Object must stay recognizable

Do

  • Let dispersion pixels extend beyond composition boundaries
  • Match pixel block colors to the source object
  • Use single objects on clean backgrounds
  • Apply consistently at 20–40% coverage

Don't

  • Apply pixel dispersion to the logo
  • Apply to UI screenshots or product interfaces
  • Scatter pixels randomly without a source object
  • Mix multiple pixel-dispersion objects in one composition

Application Guidelines

Marketing Collaterals

Social Media

  • Three-column layout as base composition
  • Twitter/X banner: 1500×500 — horizontal
  • Instagram: 1080×1080 — stacked layout
  • LinkedIn: 1200×627 — horizontal
  • Story/Reel: 1080×1920 — vertical color blocks

Presentation Slides

  • Title slides: Three-column layout + product name
  • Content: Praxeti White bg, Dark Steel text
  • Section dividers: Full-bleed Spring Green or Matte Blue
  • Data slides: White bg, brand colors for charts
  • 4px Spring Green accent bar on content slides

Print Materials

  • Business cards: Blue front / Spring Green back
  • Letterhead: White + Spring Green accent bar
  • Brochures: Three-column maps to tri-fold
  • Merchandise: Two-tone green + blue

Photography

  • Real-world objects on clean neutral backgrounds
  • High-key lighting, minimal shadows
  • Pixel-dispersion overlay in post-production
  • No lifestyle/people photography
  • No stock photos without dispersion treatment

Animation & Motion

Pixel Dispersion Animation

Assembly

800–1200ms
cubic-bezier(0.16, 1, 0.3, 1)

Pixels converge from scattered positions into the source object. Use on page load and reveals.

Disassembly

600–900ms
cubic-bezier(0.7, 0, 0.84, 0)

Object breaks apart into pixels that scatter outward. Use on exit and transitions.

Hover / Idle

3–5s continuous loop
ease-in-out

Subtle pixel drift at object edges, 2–4px movement. Ambient life.

Motion Principles

Speed

Fast and responsive. Nothing over 1200ms except continuous loops.

Easing

Always ease-out for entrances, ease-in for exits. Never linear.

Direction

Left-to-right for progress. Upward for positive values and growth.

Restraint

Reserve pixel assembly/disassembly for key moments only. Don't overuse.

Logo Animation Sequence

Step 1Icon scales up from 0 with slight bounce0–400ms
Step 2"earn" wordmark slides in from left200–800ms
Total800ms — use on splash screens, video intros, loading states

Photography & Imagery

Image Guidelines

Style

  • → Real-world objects on clean, neutral backgrounds
  • → High-key lighting, minimal shadows
  • → Isolated objects — no environmental context
  • → Pixel-dispersion applied in post-production

Color Grading

  • → Greens lean toward Spring Green
  • → Blues lean toward Matte Blue
  • → Avoid warm tones unless inherent to subject
  • → Pink/magenta acceptable for floral subjects

Do

  • Single objects on clean backgrounds
  • Consistent 20–40% dispersion coverage
  • Let pixels extend beyond boundaries
  • Match pixel colors to source object

Don't

  • Lifestyle or people photography
  • Brand color overlays or duotones on photos
  • Stock photos without dispersion treatment
  • Multiple pixel-dispersion objects in one comp
